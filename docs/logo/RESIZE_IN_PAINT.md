# How to Resize Logo in Paint (Windows)

## 🎨 Step-by-Step Guide Using Paint

---

## 📐 For splash-icon.png (200px width)

### **Step 1: Open logo.png in Paint**
1. Right-click on your `logo.png` file
2. Select "Open with" → "Paint"
   - OR open Paint first, then File → Open → Select logo.png

### **Step 2: Resize the Image**
1. Click the **"Resize"** button in the Home tab
   - OR press **Ctrl + W** (keyboard shortcut)
2. A dialog box will appear

### **Step 3: Set Size**
1. Make sure **"Pixels"** is selected (not Percentage)
2. Check the box **"Maintain aspect ratio"** ✅ (IMPORTANT!)
3. In the **Width** field, type: **200**
4. The Height will automatically adjust (keep it as is)
5. Click **"OK"**

### **Step 4: Save**
1. Click **"File"** → **"Save As"**
2. Select **"PNG picture"** from the format dropdown
3. Name it: **splash-icon.png**
4. Save it in: `assets/images/` folder
5. Click **"Save"**

---

## 📐 For favicon.png (64x64px)

### **Step 1: Open logo.png in Paint Again**
1. Right-click on your `logo.png` file
2. Select "Open with" → "Paint"
   - (Open a fresh copy, don't use the resized one)

### **Step 2: Resize the Image**
1. Click the **"Resize"** button
   - OR press **Ctrl + W**
2. A dialog box will appear

### **Step 3: Set Size**
1. Make sure **"Pixels"** is selected
2. Check the box **"Maintain aspect ratio"** ✅
3. In the **Width** field, type: **64**
4. In the **Height** field, type: **64**
5. Click **"OK"**

### **Step 4: Save**
1. Click **"File"** → **"Save As"**
2. Select **"PNG picture"** from the format dropdown
3. Name it: **favicon.png**
4. Save it in: `assets/images/` folder
5. Click **"Save"**

## 💡 Important Tips

1. **Always check "Maintain aspect ratio"** - Keeps your logo proportional
2. **Use PNG format** - Preserves transparent background
3. **Open fresh copy** - For favicon, open original logo.png again (not the resized splash-icon)
4. **Save in correct folder** - `assets/images/` folder

---

## 🚨 Common Mistakes to Avoid

❌ **Don't uncheck "Maintain aspect ratio"** - Logo will be distorted
❌ **Don't save as JPG** - Will lose transparent background
❌ **Don't use the resized splash-icon** - Open original logo.png for favicon
❌ **Don't forget to select "PNG picture"** - When saving

✅ **Do check "Maintain aspect ratio"**
✅ **Do save as PNG**
✅ **Do open fresh copy for each resize**
✅ **Do save in assets/images/ folder**
