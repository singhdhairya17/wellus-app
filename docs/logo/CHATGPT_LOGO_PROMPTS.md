# ChatGPT/GPT Prompts for Wellus Logo Design

## 🎨 Using ChatGPT Plus with DALL-E

### Step 1: Access DALL-E
1. Go to https://chat.openai.com
2. Make sure you have **ChatGPT Plus** subscription
3. DALL-E is integrated in ChatGPT (just ask for images)

### Step 2: Use These Prompts

#### Basic Logo Prompt:
```
Create a modern, minimalist logo for "Wellus" - a health and wellness mobile app. 
The logo should be:
- Square format (1:1 ratio)
- Simple and clean design
- Wellness/health theme (leaf, heart, or abstract symbol)
- Professional and modern
- Suitable for mobile app icon
- Transparent background
- High contrast colors
```

#### Detailed Logo Prompt:
```
Design a professional logo for "Wellus", a health and wellness mobile application. 

Requirements:
- Brand name: Wellus
- Industry: Health & Wellness / Nutrition / Fitness
- Style: Modern, minimalist, clean
- Format: Square (1:1), suitable for app icon
- Colors: Use a primary color (blue or green) with good contrast
- Theme: Wellness, health, vitality
- Icon elements: Consider leaf, heart, wave, or abstract geometric shapes
- Text: Can include "Wellus" text or be icon-only
- Background: Transparent
- Size: High resolution, scalable design
- Must work at small sizes (100x100 pixels)

Generate a logo that represents health, wellness, and modern technology.
```

#### Advanced Prompt with Specifics:
```
Create a premium logo for "Wellus" mobile app with these specifications:

BRAND IDENTITY:
- Name: Wellus
- Industry: Health & Wellness / Dietary Management
- Target: Health-conscious users, fitness enthusiasts

DESIGN REQUIREMENTS:
- Style: Modern, minimalist, professional
- Shape: Square format (1:1 ratio)
- Icon: Abstract wellness symbol (leaf, heart, wave, or geometric)
- Typography: If text included, modern sans-serif
- Colors: Primary blue/green, high contrast
- Background: Transparent
- Scalability: Must work from 100px to 512px

VISUAL ELEMENTS:
- Clean lines
- No gradients (solid colors)
- Simple geometric shapes
- Wellness/health symbolism
- Modern tech aesthetic

Generate a logo that feels premium, trustworthy, and wellness-focused.
```

## 🎯 Best Practices for ChatGPT Logo Prompts

### ✅ DO:
- Be specific about style (modern, minimalist, etc.)
- Mention square format (1:1 ratio)
- Request transparent background
- Specify wellness/health theme
- Ask for high resolution
- Request simple, scalable design

### ❌ DON'T:
- Don't be too vague
- Don't request complex designs
- Don't forget to mention transparent background
- Don't request too many elements
- Don't forget scalability requirements
