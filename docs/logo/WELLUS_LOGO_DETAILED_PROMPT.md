- Subtle decorative elements that enhance, not distract

- Professional and trustworthy appearance
- Wellness/health symbolism with elegant detailing
- Modern tech aesthetic with premium finishing

DETAILING ELEMENTS (Subtle & Refined):
- Subtle line work or fine strokes

- Thoughtful layering or depth
- Premium finishing touches
- Elegant composition

- Subtle texture or gradient (very light)

TECHNICAL SPECS:
- Transparent background (PNG format)
- High resolution (512x512 pixels minimum)
- Clean, crisp edges with refined details
- Works in both light and dark themes
- Details visible at 100px size

INSPIRATION:
Think premium wellness brands like Headspace, Calm, or Apple Health - 
clean base with sophisticated details, elegant refinement, premium quality.
