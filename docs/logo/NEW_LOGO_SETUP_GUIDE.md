## 🎨 Export Requirements

### **Critical: Transparent Background**

Your logo currently has a **black background** in the image. For the app, you need:

- ❌ **Current**: Black background
- ✅ **Required**: **Transparent background**

### **Export Settings:**

1. **Remove black background** - Make it transparent
2. **Format**: PNG with alpha channel
3. **Dimensions**: 1024x1024 pixels (recommended)
4. **File Size**: Under 500KB
5. **Save as**: `logo.png`

---

## 🛠️ How to Export with Transparent Background

### **If using Canva:**
1. Select your logo
2. Click "Download"
3. Format: **PNG**
4. ✅ **Check "Transparent background"** (IMPORTANT!)
5. Size: "Large" or "Original"
6. Download

### **If using Figma:**
1. Select your logo
2. Right-click → "Export"
3. Format: **PNG**
4. Make sure background layer is hidden/removed
5. Size: 2x (1024px) or 3x (1536px)
6. Export

### **If using Photoshop:**
1. Hide or delete the black background layer
2. File → Export → Export As
3. Format: **PNG**
4. ✅ Check "Transparency"
5. Dimensions: 1024x1024px
6. Export

### **If using Illustrator:**
1. Remove black background rectangle
2. File → Export → Export As
3. Format: **PNG**
4. Background: **Transparent**
5. Resolution: 300 DPI (or 72 DPI for web)
6. Dimensions: 1024x1024px
7. Export

### **If using online tools:**
- Use tools like "Remove.bg" to remove black background
- Or use "Photopea" (free online Photoshop) to remove background
- Then export as PNG with transparency
