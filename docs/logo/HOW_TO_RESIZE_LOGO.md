# How to Resize Logo for splash-icon.png and favicon.png

## 🎯 Quick Methods to Resize Your Logo

---

## Method 1: Online Tools (Easiest - No Software Needed)

### **Option A: Photopea (Free Online Photoshop)**
1. Go to: https://www.photopea.com
2. Click "File" → "Open" → Select your `logo.png`
3. Click "Image" → "Image Size"
4. For **splash-icon.png**:
   - Set width to **200px** (height will auto-adjust)
   - Click "OK"
   - Click "File" → "Export As" → "PNG"
   - Save as `splash-icon.png`
5. For **favicon.png**:
   - Open logo.png again
   - Click "Image" → "Image Size"
   - Set width to **64px** and height to **64px**
   - Make sure "Constrain Proportions" is checked
   - Click "OK"
   - Click "File" → "Export As" → "PNG"
   - Save as `favicon.png`

### **Option B: ResizeImage.net**
1. Go to: https://resizeimage.net
2. Click "Upload Image" → Select your `logo.png`
3. For **splash-icon.png**:
   - Set width to **200px**
   - Keep "Maintain aspect ratio" checked
   - Click "Resize Image"
   - Download and save as `splash-icon.png`
4. For **favicon.png**:
   - Upload logo.png again
   - Set width to **64px** and height to **64px**
   - Keep "Maintain aspect ratio" checked
   - Click "Resize Image"
   - Download and save as `favicon.png`

### **Option C: ILoveIMG**
1. Go to: https://www.iloveimg.com/resize-image
2. Click "Select images" → Choose your `logo.png`
3. For **splash-icon.png**:
   - Select "Custom size"
   - Set width to **200px**
   - Keep "Maintain aspect ratio"
   - Click "Resize image"
   - Download and save as `splash-icon.png`
4. For **favicon.png**:
   - Upload logo.png again
   - Select "Custom size"
   - Set width to **64px** and height to **64px**
   - Keep "Maintain aspect ratio"
   - Click "Resize image"
   - Download and save as `favicon.png`

## Method 2: Canva (If You Have Canva Account)

1. Go to: https://www.canva.com
2. Create new design → Custom size
3. For **splash-icon.png**:
   - Set width: **200px**, height: **200px** (or auto)
   - Upload your logo.png
   - Resize logo to fit (200px width)
   - Download as PNG
   - Save as `splash-icon.png`
4. For **favicon.png**:
   - Create new design → Custom size: **64x64px**
   - Upload your logo.png
   - Resize logo to fit (64x64px)
   - Download as PNG
   - Save as `favicon.png`

## Method 3: Windows Built-in Tools

### **Using Paint (Windows)**
1. Open `logo.png` in Paint
2. Click "Resize" button (or Ctrl+W)
3. For **splash-icon.png**:
   - Select "Pixels"
   - Set width to **200**
   - Keep "Maintain aspect ratio" checked
   - Click "OK"
   - Click "File" → "Save As" → "PNG picture"
   - Save as `splash-icon.png`
4. For **favicon.png**:
   - Open logo.png again in Paint
   - Click "Resize"
   - Select "Pixels"
   - Set width to **64** and height to **64**
   - Keep "Maintain aspect ratio" checked
   - Click "OK"
   - Click "File" → "Save As" → "PNG picture"
   - Save as `favicon.png`

### **Using Photos App (Windows 10/11)**
1. Right-click `logo.png` → "Open with" → "Photos"
2. Click "..." (three dots) → "Resize"
3. For **splash-icon.png**:
   - Select "Custom"
   - Set width to **200px**
   - Click "Save a copy"
   - Save as `splash-icon.png`
4. For **favicon.png**:
   - Open logo.png again
   - Click "..." → "Resize"
   - Select "Custom"
   - Set width to **64px** and height to **64px**
   - Click "Save a copy"
   - Save as `favicon.png`

## Method 4: Professional Tools (If You Have Them)

### **Photoshop:**
1. Open `logo.png` in Photoshop
2. Click "Image" → "Image Size"
3. For **splash-icon.png**:
   - Set width to **200px**
   - Keep "Constrain Proportions" checked
   - Click "OK"
   - File → Export → Export As → PNG
   - Save as `splash-icon.png`
4. For **favicon.png**:
   - Open logo.png again
   - Image → Image Size
   - Set width to **64px** and height to **64px**
   - Keep "Constrain Proportions" checked
   - Click "OK"
   - File → Export → Export As → PNG
   - Save as `favicon.png`

### **Figma:**
1. Import `logo.png` into Figma
2. Select the logo
3. For **splash-icon.png**:
   - Set width to **200px** (height auto)
   - Right-click → Export → PNG
   - Save as `splash-icon.png`
4. For **favicon.png**:
   - Set width to **64px** and height to **64px**
   - Right-click → Export → PNG
   - Save as `favicon.png`
