## 💡 Pro Tips

### **For Best Results:**
1. **Generate multiple variations** - Try 5-10 different images
2. **Refine prompts** - Adjust based on results
3. **Upscale** - Use high-resolution upscaling
4. **Test with overlay** - Remember dark overlay will be added
5. **Optimize** - Compress to under 2MB after generation

### **Prompt Refinement:**
- If too busy: Add "minimalist, simple, clean"
- If not peaceful: Add "zen, serene, calm, peaceful"
- If colors wrong: Specify "teal #14B8A6, green #10B981"
- If too sharp: Add "soft focus, gentle blur, subtle"
